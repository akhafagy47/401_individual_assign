## Project Description
This is a REST API web application for managing campus items (news, events, announcements). It includes:
- Backend REST API with full CRUD operations
- Persistent SQLite database storage
- Minimal frontend interface
- Docker containerization
- OpenAPI/Swagger documentation
- Cybera IPv6 deployment
